Installation

ngEasyDEV is a simple Angular Cli App and has its README.md which you can find very useful.

To get started follow this steps:

  1. Install Node and npm.
  2. Download and open ngEasyDev project.
  3. Install packages: npm install (preferred) or yarn install.

Now you have two options: serve for development or build for production

Serve for development

  1. Start project locally: ng serve or ng serve --o to open in new tab (running on port 4200).

    At this step you should remember that you can't just open index.html file and you need to start server (just use ng serve command)

  2. Make necessary changes.

Build for production

  1. Run npm run build command
  2. Put statis files from dist/ngEasyDEV to your server
  3. Use template by calling to your server
