Components

These are the examples of using template components. All components have a basic structure and configuration. Just copy a component with its styles to your project.

Most of components are based on ngx-bootstrap and some of them on bootstrap 4. You can read about all used packages in a description of components below and here.

Alerts

Alerts are based on bootstrap 4. Here is the example of our wrapper:

Input variables of AlertComponent:

Property Type Description
alertType string One of the following type: 'alert-bordered', 'alert-colored', 'alert-neutral'
alertTheme string One of the following color theme: 'info', 'primary', 'warning' or 'danger'
icon string Name of angular material icon
withIcons boolean Hide or show icon

Modals

Modals are based on ngx-bootstrap modals. Here is the example of our wrapper:

Input variables of ModalComponent:

Property Type Description
title string Title of modal
message string Main message of modal
color string One of the following colors classes: 'primary', 'success', 'warning', or 'danger'
colored boolean Simple or colored modals
header boolean Show or hide header
btn string One of the following buttons names: 'Default', 'Success', 'Warning', or 'Danger'

Progress Bars

Progress is based on ngx-bootstrap progress. Here is the example

To change progress bar style you need to add some classes to 'progress-wrap':

  1. To change color: 'progress-wrap-yellow', 'progress-wrap-violet', 'progress-wrap-pink', 'progress-wrap-blue'
  2. To change size: 'progress-wrap-small', 'progress-wrap-middle', 'progress-wrap-big'
  3. To change stripted or animated behavior you can add inputs [animated], or [striped]

You can find more examples in this component: src/app/modules/ui-elements/components/progress-bars

Tabs

Tabs is based on ngx-bootstrap tabs. Here is the example

To change tabs style you need:

  1. To add border use input 'customClass' with classes: 'bordered-top', 'bordered-bottom'
  2. To change justified behavior, use input [justified]="true" with tabset
  3. To change vertical behavior, use input [vertical]="true" with tabset and add 'tabs-vertical' class to 'tabs'

You can find more examples in this component: src/app/modules/ui-elements/components/tabs
