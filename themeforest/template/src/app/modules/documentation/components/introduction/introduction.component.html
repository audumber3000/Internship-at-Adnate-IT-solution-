Introduction

ngEasyDEV is a modern dashboard template based on Angular and Bootstrap 4 Frameworks. We’ve used modern technologies and best practices to make our product easy to work with. It’s the most convenient template for developers, because of Angular Components, clean code and detailed documentation, which allows you to build any projects easily! Use it for e-commerce, analytics, sports and other types of web or mobile applications.
